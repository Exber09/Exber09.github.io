1.1 矩阵转置

对于矩阵 AAA，定义 ATA^TAT 为 AAA 的转置，满足 Ai,jT=Aj,iA^T_{i,j}=A_{j,i}Ai,jT​=Aj,i​。

1.2 行列式

几何意义：基向量在空间中组成图形的体积

对于一个 n×nn\times nn×n 的方阵 AAA，定义其行列式 ∣A∣|A|∣A∣：

∣A∣=∑p∈Per(n)(−1)σ(p)∏i=1nAi,pi|A|=\sum\limits_{p\in Per(n)} (-1)^{\sigma(p)}\prod\limits_{i=1}^n A_{i,p_i} ∣A∣=p∈Per(n)∑​(−1)σ(p)i=1∏n​Ai,pi​​

其中 Per(n)Per(n)Per(n) 为 nnn 的排列集合，σ(p)\sigma(p)σ(p) 为 ppp 的逆序对个数，非方阵的行列式为 000。

1.2.1 相关性质

  • ∣AT∣=∣A∣|A^T|=|A|∣AT∣=∣A∣，故以下关于行的性质关于列也满足；

    每行选一个相当于每列选一个。

  • 若 AAA 不是满秩（有某一个向量可以被其他向量线性组合出来），则 ∣A∣=0|A|=0∣A∣=0；

    根据几何意义，此时基向量没有张成整个 nnn 维空间，故基向量组成图形体积为 000。

  • 交换 AAA 的任意不同两行，则 ∣A∣|A|∣A∣ 取反；

    因为交换排列任意不同两位对逆序对的改变都是奇数的。

  • ∣A∣|A|∣A∣ 是关于每一行的线性函数，即：

    • 将 AAA 中某一行整体乘上任意实数 aaa，则 ∣A∣|A|∣A∣ 会乘上 aaa；

    • ∣a+a′b+b′cd∣=∣abcd∣+∣a′b′cd∣\left|\begin{matrix}a+a'&b+b'\\c&d\end{matrix}\right|=\left|\begin{matrix}a&b\\c&d\end{matrix}\right|+\left|\begin{matrix}a'&b'\\c&d\end{matrix}\right|∣∣∣∣​a+a′c​b+b′d​∣∣∣∣​=∣∣∣∣​ac​bd​∣∣∣∣​+∣∣∣∣​a′c​b′d​∣∣∣∣​

      根据乘法分配律易得。

    根据这个性质有一个推论：

    • 将 AAA 中某一行整体乘上任意实数 aaa 再按位加到另一行，则 ∣A∣|A|∣A∣ 不变；

1.2.3 求解

注意到若一个 n×nn\times nn×n 的矩阵 AAA 只有下半部分非零，则 ∣A∣=∏i=1nAi,i|A|=\prod\limits_{i=1}^n A_{i,i}∣A∣=i=1∏n​Ai,i​。

那么类似高斯消元，通过 ∣A∣|A|∣A∣ 关于每一行的线性性进行消元，直到 AAA 只剩下下半部分为止即可。

inline int qpow(int x,int y=p-2)
{
	int res=1;
	for(;y>0;y>>=1,x=1ll*x*x%p) res=y&1?1ll*res*x%p:res;
	return res;

struct mrt
{
	int n,m,a[S][S];
	inline mrt(){n=m=0,memset(a,0,sizeof(a));}
	inline mrt(int x){n=m=x,memset(a,0,sizeof(a));}
	inline mrt(int x,int y){n=x,m=y,memset(a,0,sizeof(a));}
	inline int* operator[](int x){return a[x];}
	inline int det()
	{
		if(n!=m) return 0;
		mrt b=*this;
		int mul=1;
		for(int i=1;i<=n;i++)
		{
			for(int j=i+1;j<=n&&b[i][i]==0;j++)
			{
				if(b[j][i]!=0)
				{
					for(int k=1;k<=n;k++) swap(b[i][k],b[j][k]);
					mul=p-mul;
					break;
				}
			}
			int inv=qpow(b[i][i]);
			for(int j=i+1;j<=n;j++)
			{
				if(b[j][i]==0) continue;
				int mul=1ll*b[j][i]*inv%p;
				for(int k=1;k<=n;k++) add(b[j][k],p-1ll*b[i][k]*mul%p);
			}
		}
		for(int i=1;i<=n;i++) for(int j=1;j<i;j++) if(b[i][j]!=0) mul=0;
		for(int i=1;i<=n;i++) mul=1ll*mul*b[i][i]%p;
		return mul;
	}
};

1.3 LGV 引理

对于一个带边权的 DAG，定义一条路径的权值为路径上所有边的边权积。

对于这个 DAG 上两个大小为 nnn 的点序列 SSS 和 TTT，设 Si→TiS_i\to T_iSi​→Ti​ 的所有路径权值的和为 wi,jw_{i,j}wi,j​。

定义一个路径组为一个大小为 nnn 的排列 ppp 和大小为 nnn 的路径序列 PPP 组成的二元组 (p,P)(p,P)(p,P)，其中 PiP_iPi​ 起点为 SiS_iSi​，终点为 TpiT_{p_i}Tpi​​。定义路径组 (p,P)(p,P)(p,P) 的权值 v(p,P)v_{(p,P)}v(p,P)​ 为 PPP 中路径权值的积。

对于一个路径组 (p,P)(p,P)(p,P)，如果 PPP 中任意两条路径没有公共点，则称其为不交路径组，记这样的路径组为 (p,P)∗(p,P)^*(p,P)∗。

设 AAA 为一个 n×nn\times nn×n 的方阵，且 Ai,j=wi,jA_{i,j}=w_{i,j}Ai,j​=wi,j​，则 ∣A∣=∑(p,P)∗(−1)σ(p)v(p,P)∗|A|=\sum\limits_{(p,P)^*}(-1)^{\sigma (p)}v_{(p,P)^*}∣A∣=(p,P)∗∑​(−1)σ(p)v(p,P)∗​。

证明很简单，先乘法分配律拆开，显然 ∣A∣=∑(p,P)(−1)σ(p)v(p,P)|A|=\sum\limits_{(p,P)}(-1)^{\sigma (p)}v_{(p,P)}∣A∣=(p,P)∑​(−1)σ(p)v(p,P)​。

假若两条路径有公共点，那么把最后一个公共点后的路径交换一下，则 v(p,P)v_{(p,P)}v(p,P)​ 不变，而 (−1)σ(p)(-1)^{\sigma(p)}(−1)σ(p) 会变为相反数，故会互相抵消，证毕。

1.3.1 Cauchy–Binet 定理

对于一个 n×mn\times mn×m 的矩阵 AAA 和一个 m×nm\times nm×n 的矩阵 BBB，有：

∣AB∣=∑s∈{1,2,3,…,m},∣s∣=n∣A1,s1A1,s2…A1,snA2,s1A2,s2…A2,sn⋮⋮⋱⋮An,s1An,s2…An,sn∣×∣Bs1,1Bs1,2…Bs1,nBs2,1Bs2,2…Bs2,n⋮⋮⋱⋮Bsn,1Bsn,2…Bsn,n∣|AB|=\sum\limits_{s\in\{1,2,3,\dots,m\},|s|=n} \left|\begin{matrix}A_{1,s_1}&A_{1,s_2}&\dots&A_{1,s_n}\\A_{2,s_1}&A_{2,s_2}&\dots&A_{2,s_n}\\\vdots&\vdots&\ddots&\vdots\\A_{n,s_1}&A_{n,s_2}&\dots&A_{n,s_n}\end{matrix}\right|\times\left|\begin{matrix}B_{s_1,1}&B_{s_1,2}&\dots&B_{s_1,n}\\B_{s_2,1}&B_{s_2,2}&\dots&B_{s_2,n}\\\vdots&\vdots&\ddots&\vdots\\B_{s_n,1}&B_{s_n,2}&\dots&B_{s_n,n}\end{matrix}\right| ∣AB∣=s∈{1,2,3,…,m},∣s∣=n∑​∣∣∣∣∣∣∣∣∣​A1,s1​​A2,s1​​⋮An,s1​​​A1,s2​​A2,s2​​⋮An,s2​​​……⋱…​A1,sn​​A2,sn​​⋮An,sn​​​∣∣∣∣∣∣∣∣∣​×∣∣∣∣∣∣∣∣∣​Bs1​,1​Bs2​,1​⋮Bsn​,1​​Bs1​,2​Bs2​,2​⋮Bsn​,2​​……⋱…​Bs1​,n​Bs2​,n​⋮Bsn​,n​​∣∣∣∣∣∣∣∣∣​

也就是分别拿出 AAA 的 nnn 列，拿出 BBB 的 nnn 行，算出其行列式，乘起来。

证明很简单，构造一个三层图，头尾两层 nnn 个点，中间一层 mmm 个点。第一层第 iii 个点向第二层第 jjj 个点连 Ai,jA_{i,j}Ai,j​，第二层向第三层连 Bi,jB_{i,j}Bi,j​。

那么应用 LGV 引理不难发现等式两边都在计算从 SSS 为第一层的点，TTT 为第三层的点的 ∑(p,P)∗(−1)σ(p)v(p,P)∗\sum\limits_{(p,P)^*}(-1)^{\sigma (p)}v_{(p,P)^*}(p,P)∗∑​(−1)σ(p)v(p,P)∗​，得证。
